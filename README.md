# SparkSuite - Mini App Market

**A Java Swing-based desktop application that hosts multiple mini-applications in a modern, interactive UI.**  

SparkSuite is a level-driven mini app platform built with **Java Swing**, showcasing modern UI design principles and object-oriented programming. Currently, it includes **three fully functional mini-apps** with others planned for future updates.

---

## **Functional Mini-Apps**

### **1. Brain Spark Quiz** üß†
- A **timed quiz game** with multiple categories.  
- Users can select categories and answer questions.  
- Tracks **score** and displays results at the end.  
- Built using **Java OOP concepts** like Queues, Stacks, and HashMaps for quiz logic.  

### **2. Currency Converter** üí±
- Converts **multiple currencies** based on user input.  
- Simple and **interactive GUI** for easy usage.  
- Built with **Java Swing** components for real-time input and output display.  

### **3. Expense Tracker (BudgetBuddy)** üí∞
- Allows users to **add, track, and manage expenses**.  
- Includes **dashboard view, add expense panel, and history panel**.  
- Functional without any database, using **local Java objects** to store data.

## **Features & Highlights**

### **Modern UI/UX**
- Gradient backgrounds with soft, cohesive color palette (navy slate, blue, emerald, amber).  
- Rounded buttons with hover effects.  
- Stylish greetings, minimal vector icons, and clean footer design.  
- Modern fonts (Segoe UI / Poppins).  
- Soft shadows and card-style layout for mini-apps.  

### **Functionalities**
- Interactive **app cards** for launching mini-apps.  
- Fully operational **Java Swing GUI** without external dependencies.  
- Local storage for mini-app data where required.  

### **Technology Used**
- Java 17  
- Swing GUI components  
- OOP concepts: classes, objects, encapsulation, collections
